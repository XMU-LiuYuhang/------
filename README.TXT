
◆实验器材:
	战舰STM32F103开发板
	
◆实验目的:
	实现一个智能路灯系统，完成以下目标：
	1、实时时钟显示。并且可以修改。
	2、可以设置：”夏时”和”冬时”月份设置。如：夏时：6-9月 
	3、设置控制模式：光线控制,时间控制,手动控制。手动模式时可以通过触摸按钮控制，通过手机命令控制（选做）。
	4、设置开启路灯光线强度值，显示当前光强。
	5、路灯启停时间可以设置
	启动时间： 夏时：19：30   冬时：18:30
	关闭时间：夏时：05：30   冬时：06:30
	6、通过触摸屏实现设置内容。
	7、自己组织界面布局，及菜单形式的安排。
	8、测试：改变时间来测试程序的准确性。
	9、设置值，掉电不能丢失。
	10、显示环境温度。显示路灯的开关状态。
	11、路灯指示灯开启和关闭方式采用呼吸灯模式，渐亮和渐灭。
